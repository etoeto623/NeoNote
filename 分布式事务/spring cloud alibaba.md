spring cloud alibaba是阿里提供的一整套微服务治理框架，主要包含如下几个模块：
- api网关：[[zuul]]、[[gateway]]
- 限流降级：[[sentinel]]
- 分布式事务：[[seata]]
- 服务治理和分布式配置：[[nacos]]
- 服务监控：[[arthas]]
- 服务调用：[[dubbo]]
参考文档地址：[spring cloud alibaba知乎专栏](https://zhuanlan.zhihu.com/p/98874444)
